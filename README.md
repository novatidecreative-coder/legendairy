# Legendairy AC — Website

Premium website for **Legendairy AC**, Long Island's #1 Heating & Air Conditioning company (since 2012).

## Stack

- **Next.js 14** (App Router)
- **Tailwind CSS** + custom CSS variables
- **Google Fonts:** Bebas Neue, DM Sans, Space Mono
- **Icons:** Lucide React

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
